# SENAI ADS – Repositório Oficial da Turma de Desenvolvimento de Sistemas

# Sobre este repositório

Este repositório foi criado para apoiar as atividades práticas da turma de **Análise e Desenvolvimento de Sistemas (ADS)** do SENAI.

Ele será utilizado como ambiente colaborativo para o aprendizado de:

* Controle de versão com [Git](https://git-scm.com/)
* Trabalho em equipe utilizando [GitHub](https://github.com/)
* Organização de projetos em repositórios compartilhados
* Boas práticas de commits, branches e merges

Ao longo do curso, diversos conteúdos serão adicionados aqui, como:

* Exercícios práticos
* Projetos individuais e em grupo
* Documentações
* Materiais de estudo
* Registros de atividades da turma

O objetivo é que os alunos desenvolvam habilidades reais de colaboração em equipe, simulando o fluxo de trabalho utilizado no mercado de tecnologia.

## Organização do repositório

Este repositório poderá conter diferentes tipos de materiais organizados em pastas como:

* Projetos desenvolvidos pela turma
* Documentações e registros de atividades
* Materiais de apoio e exercícios
* Conteúdos produzidos colaborativamente
